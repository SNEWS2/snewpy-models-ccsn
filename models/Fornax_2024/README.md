* *Gravitational-wave and Gravitational-wave Memory Signatures of Core-collapse Supernovae* by Lyla Choi, Adam Burrows, and David Vartanyan, [Astrophys. J.  975:12, 2024](https://doi.org/10.3847/1538-4357/ad74f8).
